## 24 - 0: Interface 

Une interface est un type qui énumère des signatures de méthodes. Les struct de la valeur doivent implémenter ces méthodes pour que le code compile.

Les interface permettent de définir les comportements qu'une valeur doit être capable de réaliser.

```go
type Greeter interface {
    Greet() string // signature de fonction 
}
```

*`Greeter` est une interface*
