# 🧠 KatBrain

> A second brain built on open formats.

KatBrain est ma base de connaissances personnelle dédiée au développement logiciel, à l'architecture, à l'infrastructure et à la documentation de projets.

Construit avec Hugo et Hextra, il centralise mes notes techniques, mes retours d'expérience, mes documentations de projets et mes apprentissages dans un espace unique.

L'objectif est de construire un référentiel durable, versionné et facilement consultable.

---

## ✨ Features

* 📚 Documentation technique
* 🚀 Documentation de projets
* 🏗️ Notes d'architecture
* 📝 Retours d'expérience
* 🔍 Recherche intégrée
* 📊 Sommaires automatiques
* 📈 Diagrammes Mermaid
* 🌙 Compatible Dark Mode

---

## 🎯 Philosophy

KatBrain repose sur quelques principes simples :

### Open Formats

Toutes les connaissances sont stockées dans des formats ouverts :

* Markdown
* YAML
* Git

Aucun format propriétaire.

### Local First

Les données appartiennent à l'utilisateur.

Elles peuvent être :

* sauvegardées
* versionnées
* synchronisées
* hébergées librement

### Knowledge Over Tools

L'objectif n'est pas de dépendre d'un outil particulier mais de conserver et organiser durablement la connaissance.

## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/<username>/katbrain.git
cd katbrain
```

Start the development server:

```bash
hugo server
```

Open:

```txt
http://localhost:1313
```
